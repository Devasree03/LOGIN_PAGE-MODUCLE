# Login App

A modern login page built with React and Vite, styled with vanilla CSS.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Navigate to the project directory:
   ```bash
   cd login-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).

## Project Structure

- `src/components/LoginPage.jsx`: The main login form component.
- `src/index.css`: Global styles and component styling.
- `src/App.jsx`: Main application entry point.
